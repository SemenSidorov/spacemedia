Как Domino’s Pizza продавала франшизу на YouTube

Клиент
Domino’s Pizza

Международная сеть пиццерий

Канал

Видеореклама

Задача

Повысить число конверсий из CPC-размещений

Технологии

YouTube - TrueView In-Stream

Цель

Domino’s Pizza активно развивает сеть ресторанов с помощью франшиз.

Решение

Мы настроили рекламную кампанию в формате TrueView In-Stream.

бла бла бла правок до... много
  • Создание серии баннеров под каждый из продуктов зонтичного бренда; 
  • Создание уникальной серии мультфильмов для детской серии;
  • Продвижение по каждому каналу с собственным креативом. 
Стратегия воздействия: Объяснение проблематики + охват + клики. Тактика размещения: размещение на топовых охватных площадках + женские топовые площадки.

Итоговые показатели рекламной кампании:
  1. Пользователи провели на разделе клиента более 3 минут;
  2. Пользователи, просмотрели за сеанс больше 4 страниц;
  3. Досмотры видео составили 72%;
  4. Прослеживалась динамика роста продаж на период РК.
